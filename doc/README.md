# PyPDF Toolbox Documentation

This folder contains documentation for all utilities in the PyPDF Toolbox.

## Available Tools

| Tool | Description | Documentation |
|------|-------------|---------------|
| Launcher | Main launcher GUI - slim top-bar interface | [launcher/](launcher/) |
| Manual Splitter | Manually split PDFs by selecting page ranges | [manual-splitter/](manual-splitter/) |
| Text Extractor | Extract text from PDFs using Python, OCR, or Azure AI | [text-extractor/](text-extractor/) |

## Adding Screenshots

Each tool folder contains a `screenshots/` subdirectory where you can add:
- Main interface screenshots
- Feature demonstration images
- Step-by-step workflow captures

### Screenshot Naming Convention

Use descriptive names for screenshots:
- `01-main-window.png` - Main interface
- `02-file-loaded.png` - After loading a file
- `03-feature-name.png` - Specific feature demonstration

### Recommended Screenshot Settings

- **Format**: PNG (for quality) or JPG (for smaller size)
- **Resolution**: At least 1280x720 for full window captures
- **DPI**: 96 or 144 for retina displays
